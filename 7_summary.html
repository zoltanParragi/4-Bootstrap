Grid

  • rácsszerkezet kialakítás: .row, .col
  • grid fix méretek: pl. .col-6
  • Reszponzív grid: pl. .col-4 .col-md-2 .col-xl-1

  • Sor által definiált reszponzív oszlopszélesség: pl. középnézettől 2 oszlop, alatta 1 oszlop jelenik meg -> .row-cols-1 .row-cols-md-2

Form és input elemek

  • input, textarea - .form-control (megjelenésük elegánssá, kézbentarthatóvá válik)
  • Méretezésük: .form-control-lg és .form-control-sm
  • Színbeviteli mező: .form-control-color
  • .form-select, Méretezés: .form-select-lg és .form-select-sm
  • Checkbox: parent - form-check, input - class="form-check-input" type="checkbox"
  • Radio: parent - form-check, input - class="form-check-input" type="radio
  • Switch: parent - form-check form-switch, input - class="form-check-input" type="checkbox" role="switch"
  • Inline: parent - form-check form-check-inline, input - class="form-check-input" type="checkbox"
  • disabled, checked, disabled checked
  • Checkbox toggle buttons: input -type="checkbox" class="btn-check", label - class="btn btn-outline-primary"
  • Radio toggle buttons: input - type="radio" class="btn-check" name="options" id="option1" autocomplete="off" checked, label - class="btn btn-outline-secondary" for="option1"
